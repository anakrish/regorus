
- Install `wasm-pack`
  ```
  cargo install wasm-pack
  ```

- Build `regorusjs` for nodejs.
  ```
  cd bindings/wasm
  wasm-pack build --target nodejs --release
  ```

- Install [nodejs](https://nodejs.org/en/download)

- Run the test script
  ```
  $ node test.js
  ```
